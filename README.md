
# GenAI Labs & Learnings

Welcome to **GenAI**, a hands-on repository exploring practical implementations, experiments, and architectural patterns in Generative AI. This repo is designed for developers, architects, and AI enthusiasts looking to understand how GenAI tools can be applied in real-world production systems.

- Text-based prompt files for LLMs and multimodal models
